# UI-UX-Experiment2

## Aim:
The aim of this project is to design a clean and user-friendly web dashboard called LucidBites, which helps users discover and order food from ethical and sustainable restaurants.
## Algorithm:
Step 1: Start

Step 2: Load the LucidBites dashboard page.

Step 3: Display logo, search bar, user greeting, and sidebar menu.

Step 4: Show transparency report banner.

Step 5: Fetch restaurant data from database.

Step 6: For each restaurant:

Display image

Show name and category

Show price and distance

Show sustainability tag

Display “Add to Cart” button

Step 7: If user clicks “Add to Cart”

Add item to cart

Update cart count

Step 8: If user searches

Filter restaurants

Display matching results

Step 9: If user selects menu option

Show corresponding page (Near Me / Healthy / History / Settings)

Step 10: Stop (Wait for next user action).
